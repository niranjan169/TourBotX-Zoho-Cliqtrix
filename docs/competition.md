# 🏆 About the Competition — Zoho Cliqtrix Hackathon

## What is Zoho Cliqtrix?

**Zoho Cliqtrix** is a hackathon competition organized by **Zoho Corporation**, one of India's largest and most respected software companies, known globally for its comprehensive suite of business tools.

Cliqtrix challenges developers, engineers, and innovators to:
- Build complete, functional solutions exclusively using Zoho's technology stack
- Demonstrate creativity, technical depth, and problem-solving within real-world constraints
- Present innovative applications that showcase the power of Zoho's developer ecosystem

## Competition Format

Participants are expected to:
1. **Ideate** — Identify a problem that can be solved with Zoho tools
2. **Build** — Develop a working prototype using Zoho SalesIQ, Deluge scripting, Zoho CRM, or other Zoho products
3. **Integrate** — Connect Zoho with third-party APIs and platforms
4. **Present** — Demonstrate the solution live

## Why This Project?

Travel discovery is a common pain point — users often struggle to find packages relevant to their specific location. This project solves that with:
- A conversational chatbot interface (Zoho SalesIQ)
- AI-powered location intelligence (Google Gemini)
- A live, structured database of travel packages (Airtable)

The result is a seamless, intelligent travel recommendation engine built entirely within the Zoho ecosystem — exactly the type of innovative integration Cliqtrix rewards.

## Outcome

This project demonstrates the capabilities of **Zoho SalesIQ Bot Builder** and **Deluge scripting** when combined with cutting-edge AI APIs — a compelling proof that enterprise chatbots can be intelligent, personalized, and deeply integrated without complex backend infrastructure.
